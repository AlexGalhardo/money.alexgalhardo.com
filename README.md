## Setup

* Clone this repository

<!---->

```
git clone https://github.com/AlexGalhardo/money.alexgalhardo.com
```

* Enter repository

<!---->

```
cd money.alexgalhardo.com/
```

* Install dependencies

<!---->

```
bun install
```

* Setup enviroment variables

<!---->

```
cp .env.example .env
```

* Start local server

<!---->

```
bun run dev
```

* Go to: <http://localhost:5173/>
